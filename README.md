# Kivy Fishing Game

## Description
This is a simple fishing game built using the Kivy framework.

## Installation

### Prerequisites
- Python 3.6 or higher
- Kivy

### Steps

1. **Clone the repository**:
   ```sh
   git clone https://github.com/yourusername/kivy-fishing-main.git
   cd kivy-fishing-main
   ```

2. **Create a virtual environment**:
   ```sh
   python3 -m venv venv
   ```

3. **Activate the virtual environment**:
   ```sh
   source venv/bin/activate
   ```

4. **Install the required packages**:
   ```sh
   pip install kivy
   ```

## Running the Game

1. **Navigate to the project directory**:
   ```sh
   cd /Users/veronicabz/Documents/kivy-fishing-main
   ```

2. **Run the main.py file**:
   ```sh
   python main.py
   ```

This will start the Kivy application, and you should see the fishing game window appear.
